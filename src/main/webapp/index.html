API REST - Gestión de Nómina

Esta aplicación permite gestionar información de empleados usando servicios REST.

Base URL

http://localhost:8080/NominaAPI/api/nomina

Servicios disponibles

1. Obtener todas las nóminas

GET /api/nomina
Content-Type: application/json

2. Obtener nómina por ID

GET /api/nomina/{id}
Content-Type: application/json

3. Crear una nueva nómina

POST /api/nomina
Content-Type: application/json

Body:
{
  "id_empleado": 1,
  "nombre_empleado": "Juan Pérez",
  "salario": 2500.0,
  "dias_trabajados": 30
}

4. Actualizar una nómina

PUT /api/nomina/{id}
Content-Type: application/json

Body:
{
  "id_empleado": 1,
  "nombre_empleado": "Juan Pérez",
  "salario": 2800.0,
  "dias_trabajados": 28
}

5. Eliminar una nómina

DELETE /api/nomina/{id}

Formato de respuesta

Las respuestas están en formato JSON.
